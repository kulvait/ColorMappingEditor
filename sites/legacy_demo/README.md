# Legacy Demo (Vite + React + TypeScript) for Transfer Function Editor

This project is a **modernized version** of the original `transfer-function-editor-example` demo, translated to use **Vite** as the build tool, **React** as the UI library, and **TypeScript** for type safety. The purpose of this repo is to ensure that the original functionality of the Transfer Function Editor is preserved while adopting modern web development practices.
